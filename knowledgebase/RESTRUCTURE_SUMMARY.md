## How It Works

### Workflow

1. **User has mental note** → Writes markdown file
2. **Drags & drops** → Into appropriate category directory
3. **AI reads automatically** → No manual indexing needed

### Example

```
User writes: flow_state_insight_2025-01-15.md
Drops into: 06_COGNITIVE_PATTERNS/
Result: AI now has latest information
```
