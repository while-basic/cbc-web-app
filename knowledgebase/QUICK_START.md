# Quick Start Guide

## Add New Information in 3 Steps

### Step 1: Write Your Note
Create a markdown file with your mental note, insight, or update.

**Example:**
```markdown
# New Insight - January 15, 2025

Discovered that flow states correlate with...
```

### Step 2: Name It
Use format: `topic_description_date.md`

**Examples:**
- `flow_state_insight_2025-01-15.md`
- `clos_update_2025-01-15.md`
- `new_artifact_idea_2025-01-15.md`

### Step 3: Drag & Drop
Drop the file into the appropriate category directory:

- **Bio updates** → `01_BIO/`
- **Project updates** → `02_PROJECTS/[project_name]/`
- **Philosophy** → `03_PHILOSOPHY/`
- **Company info** → `04_CELAYA_SOLUTIONS/`
- **Skills/expertise** → `05_EXPERTISE/`
- **Cognitive patterns** → `06_COGNITIVE_PATTERNS/`
- **Research** → `07_RESEARCH/`
- **Music** → `08_MUSIC/`
- **Insights/terms** → `09_MENTAL_ARTIFACTS/`
- **Communication** → `10_COMMUNICATION/`

## That's It!

The AI system automatically reads all markdown files in these directories. No manual indexing needed.

---

## Need a Template?

Copy `TEMPLATE.md` from the root directory, rename it, fill it in, and drag it to the appropriate category.
